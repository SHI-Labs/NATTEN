NATTEN

Neighborhood Attention Extension

Bringing attention to a neighborhood near you!

NATTEN is an extension to PyTorch, which provides the first fast sliding window attention with efficient CPU and CUDA kernels. It provides Neighborhood Attention (local attention) and Dilated Neighborhood Attention (sparse global attention, a.k.a. dilated local attention) as PyTorch modules for both 1D and 2D data.

Install with pip

Latest release: 0.15.0

Please select your preferred PyTorch version with the correct CUDA build, or CPU build if you're not using CUDA:

  • PyTorch:

Run this command:

pip3 install natten -f https://shi-labs.com/natten/wheels/cu121/torch2.1.0/index.html

Run this command:

pip3 install natten -f https://shi-labs.com/natten/wheels/cu118/torch2.1.0/index.html

Run this command:

pip3 install natten -f https://shi-labs.com/natten/wheels/cpu/torch2.1.0/index.html

pip3 install natten -f https://shi-labs.com/natten/wheels/cu118/torch2.0.0/index.html

pip3 install natten -f https://shi-labs.com/natten/wheels/cu117/torch2.0.0/index.html

Run this command:

pip3 install natten -f https://shi-labs.com/natten/wheels/cpu/torch2.0.0/index.html

Your build isn't listed? Mac user? Just do: 

pip install natten==0.15.0

Careful though, without precompiled wheels installing might take a while.

You're also required to have CUDA > 11.0, cmake > 3.20 and PyTorch > 2.0 installed before attempting to install/build NATTEN.

Don't know your torch/cuda version? Run this: 

python3 -c "import torch; print(torch.__version__)"

Note: the CUDA version refers to your torch build, not the actual CUDA version you have installed, which may be higher.

Build from source

Once you've set up your Python environment and installed PyTorch with CUDA, simply clone and build:

Run this command:

git clone https://github.com/SHI-Labs/NATTEN
cd NATTEN
pip install -r requirements.txt
make

NOTE: NATTEN will use the PyTorch API to detect your GPU architecture, and will by default attempt to use 1/4th of the number of processes your system allows to build. You can override them by passing in the following arguments:

# Build with 2 workers/processes
make WORKERS=2

# Build targeting SM89 (Ada Lovelace)
make CUDA_ARCH="8.9"

Requirements

The latest NATTEN release only supports PyTorch, and the CUDA builds only support Kepler (SM35) and above. Make sure your GPU is supported by referring to this link.

Our GEMM-based kernels are only available for Volta (SM70) and above, and they're limited to FP16 math only on Volta and Turing (SM70 and SM75).

Usage

Easy to use, just import and go:

from natten import (
  NeighborhoodAttention1D,
  NeighborhoodAttention2D,
  NeighborhoodAttention3D,
)
na1d = NeighborhoodAttention1D(dim=128, kernel_size=7, dilation=3, num_heads=4)
na2d = NeighborhoodAttention2D(dim=128, kernel_size=7, dilation=3, num_heads=4)
na3d = NeighborhoodAttention3D(dim=128, kernel_size=7, dilation=3, num_heads=4)
